List Of HTML Tags

  1. Anchor     The anchor tag in HTML is used to create a hyperlink on the webpage.
  2. Big     The big tag in HTML is used to increase the selected text size by one larger than the surround
  3. Bold     The bold tag in HTML is used to specify the bold text without any extra importance.
  4. Button     The button tag in HTML is used to define the clickable button.Button tag is used to submit the content.
  5. Center     The center tag in HTML is used to set the alignment of text in the center. Not supported in HTML5.
  6. Div     The div tag is used in HTML to make divisions of content in the web page (text, images, header, footer, navigation bar, etc).
